# Project Aether

A web-based platform designed to help users build and host portfolio websites with ease.

## Tech Stack

- **Frontend/Backend**: Next.js
- **Database**: MongoDB
- **CSS**: Tailwind CSS
- **Testing**:
  - Unit: Jest
  - E2E: Cypress
